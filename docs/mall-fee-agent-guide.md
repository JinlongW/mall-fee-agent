## 一、业务全景：购物中心费用核算到底有多复杂？

### 1.1 费用类型矩阵

```
购物中心商户费用 = 固定费用 + 变动费用 + 特殊费用

├── 固定费用：租金(固定)、物业费、固定管理费、保险分摊
├── 变动费用：扣率租金(营业额×%)、水电费(抄表)、空调费、垃圾清运
└── 特殊费用：推广费分摊、装修保证金、进场费、违约金、免租期摊销
```

### 1.2 租金计算核心规则（Agent 价值核心）

```
模式 1：固定租金
├── 月租金 = 铺位面积 × 单价（元/㎡/月）
├── 递增：每 N 年递增 X%
└── 免租期：前 N 个月免租（物业费照收）

模式 2：扣率租金
├── 月租金 = 商户月营业额 × 扣率%
├── 业态差异：餐饮 12-18%、零售 8-15%、娱乐 15-25%
└── 保底：实际扣率 < 保底租金时，按保底收取

模式 3：两者取高（最常见！）
├── 月租金 = MAX(固定租金, 营业额 × 扣率)
└── 每月比较，取高者收费

模式 4：阶梯租金
├── 营业额 ≤ 10 万部分：扣率 8%
├── 10-30 万部分：扣率 12%
└── > 30 万部分：扣率 15% → 分段计算累加
```

### 1.3 传统痛点 = Agent 机会

| 痛点 | 具体表现 | Agent 解决 |
|------|---------|-----------|
| 规则复杂 | 不同商户合同不同规则 | AI 提取规则 → 自动适配 |
| 数据分散 | 合同在 OA、抄表 Excel、营业额 POS | Agent 统一采集 |
| 月度重复 | 每月几百户，每人规则不同 | 一键批量核算 |
| 争议频繁 | 商户质疑计算方式 | 透明化账单 + 规则追溯 |
| 对账困难 | 应收/实收/欠费人工核对数天 | 自动核对 + 差异标记 |

> 行业数据：某园区自动化后，账单制作从 **7 天 → 4 小时**，差错率降至 **0.2%**

├── 🎯 异常检测 + 争议预防（不只是算，还能发现算错的）
└── 🎯 你的行业知识 = 规则库的深度 = 护城河

## 二、Agent 设计与工作流

### 2.1 核心能力

```
✅ 做什么：
├── 合同解析 → 提取费用计算规则
├── 费用核算 → 按规则自动计算每户所有费用
├── 账单生成 → PDF + 多渠道发送
├── 差异检测 → 应收 vs 实收自动核对
├── 催缴提醒 → 到期未付自动分级催缴
├── 报表生成 → 应收/实收/欠费月报
└── 自然语言查询 → "XX 商户上月租金？"
```

### 2.2 月度工作流

```
Step 1 数据采集 → 读取合同 + 抄表 + POS 营业额 + 特殊费用
Step 2 费用核算 → 按规则计算 → 异常检测(环比>20%标记) → 人工审核
Step 3 账单生成 → PDF → 微信/短信/邮件发送
Step 4 对账催缴 → 应收vs实收核对 → 差异标记 → 分级催缴
Step 5 报表 → 月度汇总 + 楼层/业态分析 + 趋势对比
```

### 3.1 核心代码设计

#### 合同解析 → 费用规则 JSON

```json
{
  "merchant_id": "M001",
  "unit_id": "B1-023",
  "area": 120.5,
  "rent_type": "take_higher",
  "fixed_rent": { "base_price": 150, "escalation": { "interval_years": 1, "rate": 0.05 } },
  "turnover_rent": { "rate": 0.12, "minimum": 18000 },
  "free_rent_months": 2,
  "property_fee": { "price": 25 },
  "utilities": {
    "electricity": { "type": "tiered", "tiers": [{"limit":1000,"price":0.85},{"limit":3000,"price":1.05},{"above":3000,"price":1.35}], "shared_ratio": 0.15 },
    "water": { "type": "flat", "price": 4.5, "sewage": 0.5 }
  },
  "other_fees": { "marketing": 500, "pos": 200, "garbage": 300 }
}
```

#### 费用核算核心逻辑

```python
def calculate_monthly_fees(merchant, rules, period, meters, sales):
    bill = BillDetail(merchant=merchant, period=period)
    
    # 租金：固定/扣率/取高/阶梯
    fixed = rules.area * rules.fixed_rent.base_price * (1 + escalation)
    turnover = sales * rules.turnover_rent.rate
    
    if rules.rent_type == "take_higher":
        bill.rent = max(fixed, turnover, rules.turnover_rent.minimum)
    elif rules.rent_type == "tiered":
        bill.rent = calc_tiered(rules.tiers, sales)
    
    if is_free_rent_period(rules, period):
        bill.rent = 0  # 免租期，物业费照收
    
    # 物业费
    bill.property_fee = rules.area * rules.property_fee.price
    
    # 水电费（阶梯+公摊）
    bill.electricity = calc_tiered_electricity(meters["elec"], rules.utilities.electricity)
    bill.water = meters["water"].usage * rules.utilities.water.price
    
    # 其他费用
    bill.other = sum(rules.other_fees.values())
    
    # 汇总
    bill.total = bill.rent + bill.property_fee + bill.electricity + bill.water + bill.other
    
    # 异常检测（环比 > 20%）
    bill.anomalies = detect_anomalies(bill, merchant.history[-6:])
    
    return bill
```
